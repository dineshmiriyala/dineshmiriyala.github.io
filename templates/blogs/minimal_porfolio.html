Minimal Portfolio Design Language

How the site stays fast, focused, and personal

This portfolio is intentionally small on purpose. Every screen uses the same visual DNA so the story feels calm even when I’m shipping fast. Here’s the design language that keeps things consistent.

Type & Rhythm

SF Pro Display drives every page, from the landing hero to each blog post. I keep weights between 400 and 600 so the typography feels native on macOS and still sharp on Windows. Body copy floats around 1.05rem with 25px spacing, letting paragraphs breathe while still reading like product notes.

Color & Mood

The entire palette lives between #000 and #999 with white backgrounds. The contrast pushes the words forward while dropping the chrome. Accent colors don’t exist; emphasis comes from spacing, font weight, and the occasional bold tag.

Layout Rules

  • Max width locked to 800px so every post reads like a memo.
  • Global margins (60px top, 80px bottom) keep the vertical rhythm predictable.
  • Fixed Back and Home links create muscle memory across posts.
  • Social blocks stay centered with lightweight icons to finish each page.

Iconography & Micro Copy

Only three icons ship with the site: X, Blog, and Email. They show up in the hero and at the end of every article so readers always know how to reach me. Copy stays first person, present tense, and short. No filler, just the real work.

This setup makes the entire portfolio easy to maintain. Adding a new post is basically copy, paste, and write — the style system already handles the rest.
